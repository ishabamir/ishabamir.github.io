オンライン・ヘッドホン/音声テスト

ブラウザ内蔵の Web Audio API で、ヘッドホンの左右チャンネル、20Hz から 20kHz までの周波数特性、曲の BPM をその場でテストします。すべて端末内でリアルタイム合成し、録音もアップロードもせず、ページを閉じれば消えます。

3 つのテスト、開いてすぐ使える

1. 左右チャンネルテスト

左右それぞれに純音を鳴らし、左耳・右耳・ステレオ配線が正しいか、逆配線や片側無音がないかを確認します。

停止中

2. 周波数スイープ（20Hz–20kHz）

正弦波が 20Hz から 20kHz まで滑らかに上がります。聞こえる最低・最高の周波数を記録すれば、ヘッドホンの特性と自分の可聴範囲の両方が分かります。

停止中
— Hz

3. BPM タップ計測

ビートに合わせてボタンをタップすると、1 分あたりの拍数（BPM）を自動計算します。8 拍以上タップすると安定します。

—

プライバシー：すべてのテストはブラウザ内で完結します。音は Web Audio がリアルタイム合成し、タップや音声データが端末外に出ることはありません。

こんなときに使える

  • 新しいヘッドホン：まず左右テストで、逆配線や片側無音がないか確認。
  • 聴力が気になる：スイープでどこまで高域が聞こえるか確認。多くの成人は 15kHz 超で弱くなる。
  • 練習や編集：BPM タップでテンポを把握し、Soda Music で同じ速さのプレイリストを探す。

よくある質問

人の可聴周波数範囲は？

健康な若年者でおよそ 20Hz から 20kHz です。可聴上限は加齢とともに徐々に下がり、これは正常な老化現象です。

このツールは音声をアップロードしますか？

いいえ。すべての音はブラウザの Web Audio API が端末内で合成・再生します。録音もネット送信もしません。

16kHz 以上が聞こえないのは普通ですか？

とても一般的です。20〜25 歳を過ぎると上限は 15〜17kHz 程度まで下がり、正常な加齢です。

左右テストは何の役に立つ？

逆配線、片側無音、定位のずれを素早く発見でき、再生機器のステレオが正常かも確認できます。

BPM 計測は正確ですか？

タップが多いほど安定します。8 拍を超えると誤差は通常 1 分あたり±1 拍以内です。

2026 聴覚の健康スナップショット（推計）

世界保健機関（WHO）は、2050 年までに世界で約 25 億人が何らかの聴力低下を抱え、うち 7 億人超が聴覚リハビリを必要とし、10 億人超の若者が不適切な聴取（大音量・長時間）により回避可能な永続的聴力損失のリスクにさらされると推計しています。これを外挿すると、2026 年でも不適切な聴取の影響を受ける若者は 10 億人規模とみられます。

（上記は WHO の公開データから外挿した推計で、本サイトの一次実測ではありません。安全な音量とこまめな休憩を促す目的で掲載しています。）
