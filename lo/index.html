ທົດສອບຫູຟັງ ແລະສຽງອອນລາຍ

ໃຊ້ Web Audio API ໃນຕົວຂອງຕົວທ່ອງເວັບເພື່ອທົດສອບຊ່ອງທາງຊ້າຍແລະຂວາຂອງຫູຟັງຂອງທ່ານ, ກວາດການຕອບໂຕ້ຄວາມຖີ່ຈາກ 20Hz ຫາ 20kHz, ແລະແຕະ BPM ຂອງເພງ. ທຸກຢ່າງຖືກສັງເຄາະຢູ່ໃນທ້ອງຖິ່ນແບບສົດໆ — ບໍ່ມີການບັນທຶກໄວ້, ບໍ່ມີການອັບໂຫຼດ, ຖືກລຶບລ້າງເມື່ອທ່ານປິດໜ້າ.

ສາມການທົດສອບ, ພ້ອມທີ່ຈະນໍາໃຊ້

1. ການທົດສອບຊ່ອງທາງຊ້າຍ / ຂວາ

ຫຼິ້ນສຽງທີ່ບໍລິສຸດໃນແຕ່ລະດ້ານເພື່ອຢືນຢັນວ່າຫູຊ້າຍ, ຫູຂວາແລະສາຍສາຍສະເຕີລິໂອຖືກຕ້ອງ, ແລະຈັບຊ່ອງທີ່ປ່ຽນຫຼືຕາຍ.

ບໍ່ໄດ້ຫຼິ້ນ

2. ຄວາມຖີ່ກວາດລ້າ (20Hz–20kHz)

ໂຕນ sine glides ກ້ຽງຈາກ 20Hz ເຖິງ 20kHz. ໃຫ້ສັງເກດຄວາມຖີ່ຕໍ່າສຸດ ແລະສູງສຸດທີ່ເຈົ້າສາມາດໄດ້ຍິນ — ມັນທົດສອບທັງການຕອບສະໜອງຂອງຫູຟັງ ແລະໄລຍະການໄດ້ຍິນຂອງເຈົ້າເອງ.

3. BPM tap counter

ແຕະປຸ່ມພ້ອມກັບເຄື່ອງຕີ ແລະມັນຄິດໄລ່ຈັງຫວະໃນເທື່ອຕໍ່ນາທີ (BPM); ແຕະແປດເທື່ອຫຼືຫຼາຍກວ່ານັ້ນສໍາລັບການອ່ານທີ່ຄົງທີ່.

—

ຄວາມ​ເປັນ​ສ່ວນ​ຕົວ​: ທຸກ​ການ​ທົດ​ສອບ​ແລ່ນ​ຢູ່​ໃນ​ຕົວ​ທ່ອງ​ເວັບ​ຂອງ​ທ່ານ​. ສຽງຖືກສັງເຄາະໃນເວລາຈິງໂດຍ Web Audio, ແລະທັງການແຕະ ຫຼືສຽງໃດໆຂອງທ່ານອອກຈາກອຸປະກອນຂອງທ່ານ.

ແຜ່ນ cheat ແຖບຄວາມຖີ່ສຽງ

ແຖບຊ່ວງສິ່ງທີ່ທ່ານໄດ້ຍິນ / ໃຊ້
ເບດຍ່ອຍ20–60 Hzຮູ້ສຶກວ່າ rumble ຂອງເຕະແລະເບດ; ຫຼາຍເກີນໄປເຮັດໃຫ້ຂີ້ຕົມ.
ເບສ60–250 Hzຮ່າງກາຍຂອງຈັງຫວະແລະພະລັງງານ; ບອກເຈົ້າວ່າເບສແໜ້ນບໍ່.
ກາງຕ່ຳ250–500 Hzຄວາມອົບອຸ່ນຂອງສຽງແລະເຄື່ອງມື; boxiness ອາໃສຢູ່ທີ່ນີ້.
ກາງ500 Hz–2 kHzຮ່າງກາຍ vocal ຕົ້ນຕໍແລະການປະກົດຕົວ; ຂັບຄວາມສະດວກສະບາຍໃນການຟັງດົນນານ.
ກາງ​ສູງ2–4 kHzຄວາມແຈ່ມແຈ້ງ ແລະ ຄວາມຊັດເຈນ; harsh ແລະ piercing ຖ້າ overdone.
ປະກົດຕົວ4–6 kHzລາຍລະອຽດແລະຄວາມລະອຽດ; ໂຄງສ້າງຂອງເຄື່ອງມື.
ຄວາມສະຫວ່າງ6–20 kHzອາກາດແລະ sparkle; ຊ່ວງທໍາອິດທີ່ສູນເສຍໄປກັບອາຍຸ.

ໃນເວລາທີ່ມັນມາຢູ່ໃນ handy

  • ຫູຟັງໃໝ່: ທົດສອບຊ້າຍ/ຂວາກ່ອນເພື່ອຢືນຢັນວ່າບໍ່ມີຫຍັງຖືກປ່ຽນ ຫຼືຕາຍຢູ່ຂ້າງໜຶ່ງ.
  • ສົງໃສວ່າສູນເສຍການໄດ້ຍິນ: ໃຊ້ກວາດເພື່ອເບິ່ງວ່າເຈົ້າຍັງໄດ້ຍິນຢູ່ສູງເທົ່າໃດ — ຜູ້ໃຫຍ່ສ່ວນໃຫຍ່ຈະຫາຍໄປເກີນ 15kHz.
  • ການປະຕິບັດຫຼືການແກ້ໄຂ: ແຕະອອກ BPM ເພື່ອແກ້ໄຂ tempo, ຫຼັງຈາກນັ້ນຊອກຫາລາຍການຫຼິ້ນດ້ວຍຄວາມໄວດຽວກັນໃນ Soda Music.

ຄຳຖາມທີ່ມັກຖາມເລື້ອຍໆ

ລະດັບຂອງການໄດ້ຍິນຂອງມະນຸດແມ່ນຫຍັງ?

ປະມານ 20Hz ຫາ 20kHz ສໍາລັບໄວຫນຸ່ມທີ່ມີສຸຂະພາບດີ. ຂີດຈຳກັດດ້ານເທິງທີ່ໄດ້ຍິນໄດ້ຫຼຸດລົງເທື່ອລະກ້າວຕາມອາຍຸ, ເຊິ່ງເປັນສ່ວນປົກກະຕິຂອງອາຍຸ.

ເຄື່ອງມືນີ້ອັບໂຫລດສຽງຂອງຂ້ອຍບໍ?

ບໍ່. ສຽງທັງໝົດຖືກສັງເຄາະ ແລະຫຼິ້ນຢູ່ໃນທ້ອງຖິ່ນໂດຍ Web Audio API ຂອງຕົວທ່ອງເວັບ; ບໍ່ມີຫຍັງຖືກບັນທຶກ ຫຼືສົ່ງຜ່ານເຄືອຂ່າຍ.

ມັນເປັນເລື່ອງປົກກະຕິທີ່ຈະບໍ່ໄດ້ຍິນເກີນ 16kHz?

ທົ່ວໄປຫຼາຍ. ສໍາລັບຄົນສ່ວນໃຫຍ່ທີ່ມີອາຍຸ 20 ຫາ 25 ປີ, ຂອບເຂດຈໍາກັດເທິງຫຼຸດລົງຢູ່ທີ່ປະມານ 15 ຫາ 17kHz - ອາຍຸປົກກະຕິ.

ການທົດສອບຊ້າຍ / ຂວາແມ່ນຫຍັງດີ?

ມັນ​ເປີດ​ເຜີຍ​ຊ່ອງ​ທາງ​ທີ່​ຖືກ​ສະ​ຫຼັບ​ຢ່າງ​ວ່ອງ​ໄວ​, ຮູບ​ພາບ​ຂ້າງ​ຕາຍ​ຫຼື​ນອກ​ສູນ​, ແລະ​ຢືນ​ຢັນ​ການ​ສະ​ເຕ​ຣິ​ໂອ​ອຸ​ປະ​ກອນ​ຂອງ​ທ່ານ​ເຮັດ​ວຽກ​.

ຕົວນັບ BPM ຖືກຕ້ອງເທົ່າໃດ?

ຫຼາຍທ່ານປາດສະຫມໍ່າສະເຫມີມັນໄດ້ຮັບ; ທີ່ຜ່ານມາແປດຕີຄວາມຜິດພາດແມ່ນປົກກະຕິແລ້ວພາຍໃນບວກຫຼືລົບຫນຶ່ງ BPM.

ພາບລວມຂອງການໄດ້ຍິນ-ສຸຂະພາບປີ 2026 (ຄາດຄະເນ)

ອົງການ​ອະນາ​ໄມ​ໂລກ (WHO) ຄາດ​ວ່າ​ຮອດ​ປີ 2050 ເກືອບ 2,5 ຕື້​ຄົນ​ຈະ​ມີ​ການ​ສູນ​ເສຍ​ການ​ໄດ້ຍິນ​ບາງ​ລະດັບ, ​ໃນ​ນັ້ນ 700 ລ້ານ​ກວ່າ​ຄົນ​ຈະ​ຕ້ອງ​ໄດ້​ຮັບ​ການ​ຟື້ນ​ຟູ​ການ​ໄດ້ຍິນ; ຜູ້ໃຫຍ່ໄວໜຸ່ມຫຼາຍກວ່າ 1 ຕື້ຄົນມີຄວາມສ່ຽງຕໍ່ການສູນເສຍການໄດ້ຍິນແບບຖາວອນ, ຫຼີກເວັ້ນຈາກການຟັງທີ່ບໍ່ປອດໄພ (ດັງເກີນໄປ, ດົນເກີນໄປ). Extrapolating, ປະຊາກອນຫນຸ່ມທີ່ໄດ້ຮັບຜົນກະທົບຈາກການຟັງທີ່ບໍ່ປອດໄພໃນປີ 2026 ຍັງຄົງຢູ່ໃນລໍາດັບຂອງຫນຶ່ງຕື້.

(ຕົວເລກຂ້າງເທິງນີ້ແມ່ນການປະເມີນທີ່ສ້າງຂຶ້ນຈາກຂໍ້ມູນສາທາລະນະຂອງ WHO, ບໍ່ແມ່ນການວັດແທກດ້ວຍມືທຳອິດໂດຍເວັບໄຊນີ້; ສະແດງໃຫ້ເຫັນເພື່ອຊຸກຍູ້ປະລິມານທີ່ປອດໄພ ແລະ ການພັກຜ່ອນເປັນປະຈຳ).

ທີ່ກ່ຽວຂ້ອງ

ຕ້ອງການທົດສອບໃນຂະນະທີ່ທ່ານຟັງ, ຫຼືໃຊ້ tempo ວັດແທກສໍາລັບລາຍການຫຼິ້ນຂອງທ່ານບໍ? ມຸ່ງຫນ້າໄປ ດາວໂຫລດ Soda Music ສໍາລັບ desktop ເພື່ອໃຫ້ໄດ້ຮັບລູກຄ້າ desktop.
